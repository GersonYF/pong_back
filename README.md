# pong_back
Pong Django Backend

# Install steps

## Install Virtualenv
pip virtualenv

## Create Virtual Env
virtualenv pong

## Activate Env
source pong/bin/activate

## Install Requirements
pip install -R requirements.txt

## Run Server
python manage.py runserver
